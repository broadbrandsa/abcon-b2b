# Group company logos

These `*.svg` files are **placeholder wordmarks** generated for the About page so it
renders immediately. They are NOT the official brand logos.

To use the real logos, download each company's official logo and replace the matching
file (keep the same filename), or update the `logo` path in
`src/content/group-companies.tsx`.

Sources: the Abcon Group site and each company's own site —
https://abcongroup.co.za/group-companies/ , abcondev.co.za, crafthomes.co.za,
striveprop.co.za, tadvest.com, and the individual company pages.
